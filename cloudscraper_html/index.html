What CloudScraper can do

Sites Listing

The app collects and presents all sites of the logged-in user’s tenant they have access to. The list includes data about each site such as name, URL, description, last modified and creation time.

Search

In order to find the site your user is looking for (even faster) they have the option to use the search functionality!

Performance

Performance is an important factor in applications. CloudScraper was built in a way to provide you premium service, taking almost no time to load your results.

Data

We at CloudWerk value privacy a lot. Therefore, we don’t save any data about the user or their respective tenant.

Decide the platform

CloudScraper can be used as a full-page web part in SharePoint Online but also as a Teams tab. Both come in one package, so you decide where you want to use it. Alternatively you can also use the web application.
